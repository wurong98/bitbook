## 钱包概述

### 什么是钱包

```
钱包不是"存储比特币的地方"

正确理解：
  1. 管理私钥的工具
  2. 构造和签署交易的软件
  3. 显示余额和历史的界面

比特币存储在区块链上
钱包只保管访问权限（私钥）
```

### 钱包功能

```
核心功能：
  1. 生成密钥对（私钥/公钥）
  2. 管理地址
  3. 构造交易
  4. 签署交易
  5. 广播交易

辅助功能：
  1. 显示余额
  2. 追踪历史
  3. 地址簿
  4. 备份和恢复
```

## 热钱包

### 定义

```
热钱包 = 私钥存储在联网设备

特点：
  1. 私钥在互联网连接的设备上
  2. 方便快速支付
  3. 容易受到恶意软件威胁
```

### 类型

#### 桌面钱包
```
运行在个人电脑上

优点：
  用户完全控制
  无需信任第三方

缺点：
  需要管理备份
  容易感染恶意软件
  需要同步区块链（全节点）或依赖服务器

例：Bitcoin Core、Electrum
```

#### 移动钱包
```
运行在手机上

优点：
  便携
  日常使用方便

缺点：
  手机被盗 = 比特币被盗
  屏幕小难以验证地址
  存储空间有限

例：Trust Wallet、BlueWallet
```

#### Web 钱包
```
运行在网页浏览器

优点：
  无需下载安装
  随处可用

缺点：
  极度不安全
  网站被黑可能丢失比特币
  依赖网站运营者

例：blockchain.com、coinbase.com（交易所）
```

## 冷钱包

### 定义

```
冷钱包 = 私钥离线存储

特点：
  1. 私钥从不接触互联网
  2. 非常安全
  3. 不方便经常使用
  4. 适合长期存储
```

### 类型

#### 硬件钱包
```
专用设备存储私钥

工作原理：
  1. 交易在主机构造
  2. 传输到硬件钱包显示
  3. 用户在设备上确认
  4. 设备内部签名
  5. 签署的交易返回主机
  6. 主机广播

优点：
  极其安全
  私钥永不离开设备

缺点：
  需要购买（$50-200）
  需要学习使用

例：Ledger Nano S、Trezor
```

#### 纸钱包
```
将私钥打印在纸上

生成方式：
  使用离线设备生成
  打印下来

优点：
  完全离线
  一次性成本为零
  容易理解

缺点：
  容易损坏或丢失
  恢复复杂
  已过时（现在用 BIP 39）
```

#### 脑钱包
```
从记忆密语生成私钥

方式：
  记住一句话
  用它生成密钥

缺点：
  容易被暴力破解
  人类记忆不够随机
  不推荐使用
```

## 托管钱包

### 定义

```
托管钱包 = 第三方保管私钥

特点：
  1. 用户无法访问私钥
  2. 交易由第三方签署
  3. 依赖第三方的诚实性
  4. 第三方可以冻结账户
```

### 类型

#### 交易所
```
提供商：Coinbase、Kraken、币安

用途：
  购买/交易比特币
  不是长期存储方案

风险：
  1. 交易所被黑
  2. 交易所失败（如 FTX）
  3. 政府冻结
  4. 用户隐私暴露
```

#### 托管服务
```
专业第三方保管

服务商：Fidelity Custody、Coinbase Custody

用途：
  机构级别存储
  大额资金

费用：
  高（通常 10-50 bps/年）

安全性：
  比交易所高（专业保管）
  但仍有风险
```

## 钱包选择

### 选择标准

```
小额日常使用：
  → 移动热钱包或轻钱包
  → 如 Trust Wallet、BlueWallet

中等金额：
  → 桌面钱包或硬件钱包
  → 如 Electrum、Ledger

大额长期存储：
  → 硬件钱包 + 多重签名
  → 如多个 Ledger 或 Trezor

机构级别：
  → 专业托管服务
  → 如 Fidelity、Coinbase Custody
```

### 安全性等级

```
最安全到最不安全：

1. 多重签名冷钱包
   多个硬件钱包
   需要 m-of-n 签名
   
2. 硬件钱包（单个）
   Ledger、Trezor
   
3. 离线生成的纸钱包
   （假设正确保存）
   
4. 桌面钱包（Electrum）
   运行在安全的电脑上
   
5. 移动钱包
   防护能力有限
   
6. Web 钱包
   需要信任运营者
   
7. 交易所账户
   第三方完全控制
   不该作为钱包
```

## 常见问题

### Q1: 我应该选哪种钱包？

**A**: 取决于：
1. 金额大小
2. 使用频率
3. 安全需求
4. 技术水平

### Q2: 热钱包真的不安全吗？

**A**: 
- 相对而言，是的
- 但定期小额使用可以接受
- 不应存放大额资金

### Q3: 硬件钱包会不会过时？

**A**: 
- 不会，基于成熟密码学
- 私钥格式不变
- 可长期使用

### Q4: 忘记 pin/密码怎么办？

**A**: 
- 如果有种子词：可恢复
- 如果没有：可能丢失访问权限
- 总是记住种子词备份！

### Q5: 能否改变钱包类型？

**A**: 
- 可以导入私钥
- 或从种子词恢复
- 硬件钱包支持导出
- 保证可迁移性
